🏦 Trilha — Sistemas Agênticos com LangChain4j + Micronaut 5 + Java 25

Do modelo mental ao agente em produção, sempre ancorado no assistente de chat de um banco que consulta produtos via ferramentas.

Missão — construir, em produção, o backend agêntico de um assistente bancário (boleto, DDA, Pix, Super Extrato, cartão/fatura, investimentos, financiamento, conta nacional e global) em Claude Sonnet (200K) via AWS Bedrock, acertando arquitetura, segurança e escala. Ver MISSION.md.

Parte 3 — Micronaut 5 + Java 25

Parte 4 — Produção, Segurança e Escala

  1. Lição 15 · Segurança: prompt injection e o OWASP LLM Top 10 (2025) Direta vs. indireta (RAG); o mapa OWASP 2025 → defesa do banco; o controle vive no código, nunca no prompt.

  5. Lição 19 · O ecossistema honesto: Java vs. Python Real em Java (LangChain4j/Spring AI GA; agentic beta) vs. só Python (LangGraph, Deep Agents); LangServe arquivado; OTel para LangFuse/LangSmith.
